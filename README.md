# vue-3-auth-jwt

Requires running Node.js [server](https://github.com/ahmedovv123/JWT-Auth-NodeJS) 

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
